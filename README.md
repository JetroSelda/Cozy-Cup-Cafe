# Project Description
Cozy Cup Cafe is a Coffee shop that serve variety of drinks and pastries.

### Features
- Hot Beverages
-  Milk Teas
-  Fruit Tea
-  Pure Tea
-  Smoothies
-  Frappes
-  Cold Beverages
-  CozyCup's Food
-  Rice Mea
-  Pastas
-  Dessert
